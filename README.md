# 🏠 House Rent Prediction - Machine Learning Model

## 📌 Project Overview
A Machine Learning model to predict house rent based on Size, BHK, and Bathrooms using Linear Regression.

## 🛠️ Tech Stack
- Python, Pandas, NumPy
- Scikit-learn (Linear Regression)
- Matplotlib, Seaborn (EDA & Visualization)

## 📊 What's Inside
- Exploratory Data Analysis (EDA)
- Correlation Heatmap
- Model Training & Evaluation (MAE, MSE, R²)
- Actual vs Predicted Visualization
- Custom Rent Prediction
